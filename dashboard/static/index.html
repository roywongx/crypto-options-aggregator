      📚 如何解读此图

      橙色线 (左轴) = 痛点曲线，最低点即最大痛点，到期时价格常被吸引向此收敛。

      柱状图 (右轴) = OI 净敞口 (Gamma 效应)：绿色柱=Call OI > Put OI(多头 Gamma 区)→波动受抑制；红色柱=Put OI > Call OI(空头 Gamma 区)→波动易放大。

      ⭐ Gamma Flip 点（红绿交界）是多空分界线，突破后波动模式可能剧变！

      垂直虚线：绿色=Gamma Flip，橙色=最大痛点，白色=现货价格。

      💡 Gamma Flip 深度解析

      什么是 Gamma Flip？Gamma Flip 是 Call 和 Put 的 Gamma 敞口互相转换的临界点。在此价格水平，做市商的 Gamma 对冲压力从"抑制波动"转变为"放大波动"。

      为什么重要？当价格高于 Gamma Flip：机构多头主导，期权商高抛低吸，波动受抑制。当价格低于 Gamma Flip：机构空头主导，期权商追涨杀跌，波动易放大。

      实战用法：①判断突破时机——突破 Gamma Flip 后波动可能剧烈放大；②选择策略——在多头 Gamma 区更适合卖出期权收权利金；③风险管理——在空头 Gamma 区注意止损和仓位控制。
